# Mycalc

基于 C++ 和 EasyX 图形库的桌面计算器，支持带括号的四则运算。

## 功能

- 四则运算：加、减、乘、除
- **括号表达式**：正确处理带括号的算式（中缀转后缀求值）
- 一元负号（如 `-5`、`3*(-2)`）
- 小数输入与计算
- Del 退格删除 / C 一键清空
- 除零检测与括号匹配错误提示

## 修复说明

原版计算器在处理带括号的表达式时计算结果有误。本版本重写了表达式求值引擎：

- 使用**中缀表达式转后缀表达式（逆波兰表达式）**算法，严格遵循运算符优先级
- 正确处理括号嵌套与括号不匹配的情况
- 支持一元负号（例如 `(-3)+5`、`3*(-2+1)`）

## 界面美化

- 深色主题背景，按钮按功能**分色显示**：
  - 红色 — 清空 / 删除
  - 绿色 — 等号
  - 蓝色 — 运算符（+ - * /）
  - 深灰 — 数字
- Consolas 等宽字体，算式与结果显示清晰
- 窗口标题 "Calculator"

## 环境要求

- Visual Studio 2022（或 2019），安装 **"使用 C++ 的桌面开发"** 工作负载
- [EasyX 图形库](https://easyx.cn) — 下载安装即可

## 运行方法

1. 克隆仓库：
```bash
git clone https://github.com/yaoqi1018/Mycalculator.git
```

2. 用 Visual Studio 打开 `Mycalc/Mycalc.sln`
3. 选择 **x64** 平台，按 **F5** 编译运行

## 操作说明

| 按钮 | 功能 |
|------|------|
| `0-9` | 输入数字 |
| `.` | 小数点 |
| `+ - * /` | 运算符 |
| `( )` | 左右括号 |
| `=` | 计算结果 |
| `C` | 清空全部 |
| `Del` | 退格删除 |

## 项目结构

```
Mycalc/
├── main.cpp           # 程序入口，初始化窗口
├── calculator.h/cpp   # 全局变量与常量定义
├── calc.h/cpp         # 计算逻辑（输入处理、中缀转后缀、求值）
├── display.h/cpp      # 界面绘制（按钮、文字、颜色）
├── input.h/cpp        # 鼠标点击事件处理
└── Mycalc.sln         # Visual Studio 解决方案
```
